# Memory Game

> This application is made with ExpressJs and PostgreSQL

## Get starting

- Clone the project

```sh
git clone git@github.com:GermainMichaud/memory-game.git
cd ./memory-game
```

- Install dependencies

```sh
npm i # or npm install
```

- Run the application

```sh
npm run dev # in dev mode (with hot reload)
# or
npm start
```
